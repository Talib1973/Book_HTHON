# Module 3: NVIDIA Isaac Sim

## Overview

NVIDIA Isaac Sim provides GPU-accelerated robotics simulation with photorealistic rendering, enabling large-scale synthetic data generation for AI training.

## What You'll Learn

### Week 8: Isaac Sim Fundamentals
- Set up Isaac Sim with Omniverse
- Create USD-based robot models
- Use GPU ray-tracing for sensors

### Week 9: Synthetic Data Generation
- Generate annotated training datasets
- Domain randomization techniques
- Sim-to-real transfer

### Week 10: Isaac Gym Integration
- Train RL policies with Isaac Gym
- Parallelize thousands of environments
- Deploy trained models to ROS 2

## Learning Objectives

1. Leverage GPU acceleration for physics simulation
2. Generate synthetic training data at scale
3. Train reinforcement learning policies efficiently
4. Transfer sim-trained models to real robots
